# Challenge-Alura-TelecomX
Análisis de evasión de clientes

🚀 Descripción del Proyecto

Este proyecto desarrolla un análisis completo de evasión de clientes (Churn) en una empresa de telecomunicaciones utilizando técnicas de limpieza de datos, análisis exploratorio y visualización.

El objetivo principal es identificar patrones de comportamiento asociados a la cancelación del servicio, permitiendo generar estrategias orientadas a mejorar la retención de clientes.

🎯 Objetivos

✔ Analizar la distribución de clientes que cancelan el servicio
✔ Detectar factores asociados a la evasión
✔ Limpiar y preparar datos reales para análisis
✔ Generar visualizaciones claras y profesionales
✔ Proponer recomendaciones estratégicas basadas en datos

🧹 Limpieza y Preparación de Datos

Durante el proceso se realizaron las siguientes tareas:

Normalización de datos anidados (JSON → columnas estructuradas)

Identificación y tratamiento de valores faltantes

Corrección de tipos de datos inconsistentes

Estandarización de variables categóricas

Creación de la variable Cuentas_Diarias a partir de la facturación mensual

Esto permitió trabajar con un dataset confiable y listo para el análisis.

📈 Análisis Exploratorio de Datos (EDA)

Se exploraron diferentes perspectivas del comportamiento del cliente:

🔹 Distribución de Evasión

Visualización general del porcentaje de clientes que permanecen vs. cancelan.

🔹 Variables Categóricas

Análisis de evasión según:

Género

Tipo de contrato

Método de pago

Servicio de internet

🔹 Variables Numéricas

Comparación entre clientes que cancelan y los que permanecen usando:

Tiempo de permanencia (tenure)

Facturación mensual

Facturación total

Cuentas diarias

💡 Insights Principales

📉 Los clientes con contratos mes a mes presentan mayor evasión.

⏱ La mayoría de cancelaciones ocurre en los primeros meses.

💰 Cargos mensuales elevados se asocian con mayor abandono.

👥 Las variables demográficas tienen menor impacto que las condiciones del servicio.

🧠 Recomendaciones Estratégicas

Incentivar contratos de mayor duración.

Implementar estrategias de fidelización temprana.

Optimizar planes con alto costo mensual.

Detectar clientes en riesgo durante las primeras etapas del servicio.
